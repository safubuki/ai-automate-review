# はじめに

## このドキュメント群について

本ドキュメント群は、AIを活用したPRレビュー自動化のサンプル環境として作成されたものです。

実際のチームで運用マニュアルを管理する場面を想定し、ドキュメント特有の品質基準をAIレビューで担保する仕組みを体験できます。

## 対象読者

- 社内マニュアルや手順書を作成・メンテナンスする担当者
- GitHubを活用したドキュメント運用を検討しているチーム
- AIレビューを業務プロセスに取り入れたいと考えている方

## 本リポジトリの特徴

- GitHub公式機能（Copilot Code Review / GitHub Models）のみを使用
- 外部ツールへの依存を極力排除
- ドキュメントレビューに特化した指示・プロンプトを公開
